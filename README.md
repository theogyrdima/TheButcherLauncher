## 📥 Как установить? (Очень просто!)

### 1️⃣ Скачай лаунчер

- Зайди в раздел [**Releases**](https://github.com/theogyrdima/TheButcherLauncher/releases)
- Возьми последнюю версию
- Распакуй архив куда-нибудь (куда вам удобнее)

### 2️⃣ Запусти

```bash
TheButcherLauncher.exe
```

### 3️⃣ Выбери профиль и вперёд на сервер! 🎮

Всё просто, правда?

## 🐛 Что-то не работает? Помогу!

Если у тебя возникла проблема, не волнуйся — её можно решить!

| Что случилось? | Что делать? |
|---------|---------|
| Лаунчер не запускается | Проверь, установлен ли .NET 8 Runtime |
| Не подключается к серверу | Проверь интернет (должен быть стабильным) |
| Мало FPS в игре | Понизь графику в настройках лаунчера |
| Java не найдена | Установи Java 8+ или используй встроенный проверяльщик |

Если ничего не помогло — [создай Issue](https://github.com/theogyrdima/TheButcherLauncher/issues), мы разберёмся вместе! 💪
