## ✨ Что тебя ждёт?

Привет, братан! 👋 Вот че топового я сделал:

- 🚀 **Быстрый запуск** — Мгновенная загрузка Minecraft без мучений
- 🔄 **Умное обновление** — Скачиваются только новые файлы, чтобы не занимать место
- ⚡ **Лёгкий лаунчер** — Собственная разработка на передовых технологиях
- 🎨 **Красивый интерфейс** — Приятно смотреть и удобно использовать
- 🔧 **Полная настройка** — Выделение памяти, профили, всё под твои нужды
- 👤 **Microsoft-аккаунты** — Авторизуйся через свою учётку
- 📦 **Поддержка модов** — NeoForge встроен, все моды работают из коробки

## 📥 Как поставить?

### 1️⃣ Скачай лаунчер

- Зайди в раздел [**Releases**](https://github.com/theogyrdima/TheButcherLauncher/releases)
- Там нажми на ласт версию **TheButcherLauncher.exe**
- Закинь файл куда угодно (советую в ProgramFiles чтобы потом сделать ярлык)

### 2️⃣ Запусти

```bash
TheButcherLauncher.exe
```

### 3️⃣ Зарегайся в свой майкрософт или в оффлайн-аккаунт! 🎮

Плей ин оффлайн мод:
- ■ Аккаунты
- ▼ Введи ник
- ▼ Войди с оффлайн-аккаунтом
- [✕] **Готово**

Играешь с лицензией?
В том же окне нажми **[Войти через Microsoft]**

## 🐛 Обкашлять вопросик

Произошла какая-то херь или надо просто обкашлять вопросик, создай так называемый **[Issue](https://github.com/theogyrdima/TheButcherLauncher/issues)**

Ну если в падлу то вот ответы на частые запросы:
| Что случилось? | Что делать? |
|---------|---------|
| Лаунчер не запускается | Проверь, установлен ли .NET 8 Runtime |
| Не подключается к серверу | Проверь интернет или статус сервера |
| Мало FPS в игре | Поменяй настройки в майне или выдели больше ОЗУ |
| Java не найдена | Значит надо **[скачать Java](https://javadl.oracle.com/webapps/download/AutoDL?BundleId=253456_ba687cb3cbb24342adc8fdf890b993dc)** |
