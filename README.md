## ✨ Основные возможности

- 🚀 **Быстрый запуск** — Оптимизированный процесс загрузки Minecraft
- 🔄 **Автоматическое обновление** — Загружаются только изменённые файлы
- ⚡ **Эффективный лаунчер** — Собственная реализация на современных технологиях
- 🎨 **Красивый интерфейс** — Современный дизайн на WPF
- 🔧 **Настраиваемость** — Выделение памяти, профили, параметры запуска
- 👤 **Microsoft-аккаунты** — Авторизация через учётные записи Microsoft
- 📦 **Поддержка NeoForge** — Встроенная работа с модами

## 📥 Быстрая установка

### 1️⃣ Скачайте лаунчер

- Перейдите в раздел [**Releases**](https://github.com/theogyrdima/TheButcherLauncher/releases)
- Скачайте последнюю версию
- Распакуйте архив в удобную папку

### 2️⃣ Запустите

```bash
TheButcherLauncher.exe
```

### 3️⃣ Выберите профиль и играйте!

## 🐛 Возникли проблемы?

| Проблема | Решение |
|---------|---------|
| Лаунчер не запускается | Проверьте наличие .NET 8 Runtime |
| Ошибка подключения к серверу | Убедитесь в стабильности интернета |
| Низкий FPS в игре | Уменьшите параметры графики в настройках лаунчера |
| Java не найдена | Установите Java 8+ или используйте встроенную проверку |

Если проблема не решена — [создайте Issue на GitHub](https://github.com/theogyrdima/TheButcherLauncher/issues)
