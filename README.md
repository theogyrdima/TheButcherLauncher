# TheButcherLauncher

Современный лаунчер Minecraft, разработанный специально для сервера "Коч братан".

> ✅ Основы лаунчера готовы.

## ✨ Возможности

- 🚀 Быстрый запуск Minecraft
- 🔄 Автоматическое обновление файлов
- ⚡ Загрузка только изменившихся файлов
- 🎨 Современный интерфейс на WPF
- 🔧 Настраиваемое выделение оперативной памяти
- ☕ Автоматическая проверка и установка Java (в разработке)
- 👤 Поддержка Microsoft-аккаунтов

## 📥 Установка

1. Перейди в раздел **Releases**.
2. Скачай последнюю версию лаунчера.
3. Распакуй архив в любую папку.
4. Запусти `TheButcherLauncher.exe`.
